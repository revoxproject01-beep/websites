# NexusWealth - Modern AI-Powered Wealth Management Platform

A complete, professional wealth management website template built with HTML, Tailwind CSS, Font Awesome, and vanilla JavaScript with Chart.js.

## Features

- **Modern, Responsive Design** - Works on all device sizes
- **RTL Support** - Right-to-Left layout toggle with localStorage persistence
- **11 Complete Pages**:
  - Home Version 1 (index.html)
  - Home Version 2 (index2.html)
  - About Us (about.html)
  - Services (services.html)
  - Pricing (pricing.html)
  - Contact (contact.html)
  - FAQ (faq.html)
  - Login (login.html)
  - User Dashboard (user-dashboard.html)
  - Admin Dashboard (admin-dashboard.html)
  - 404 Page (404.html)
  - Coming Soon (coming-soon.html)

## Technologies Used

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Chart.js for data visualization
- Vanilla JavaScript (no frameworks)

## Color Scheme

## Key Functionality

### Navigation
- Responsive navbar with dropdown menus
- Sticky navigation with shadow on scroll
- Mobile-friendly hamburger menu

### RTL Toggle
- Globe icon in navbar to switch between LTR/RTL layouts
- Preference saved in localStorage

### Dashboard Features (User & Admin)
- Fixed top navbar with notifications and RTL toggle
- Collapsible sidebar with smooth transitions
- Responsive design (sidebar hidden on mobile by default)
- Chart.js data visualization
- Countdown timer
- Notification system

### FAQ Page
- Collapsible accordion sections
- Search functionality to filter questions
- Smooth expand/collapse animations

### Contact Page
- Interactive Google Maps embed
- Business hours display
- Contact form with validation

## Customization

To customize the template:

1. **Colors**: Modify the color classes in HTML files or override in css/styles.css
2. **Content**: Edit the HTML content directly
3. **Images**: Replace placeholder images with your own
4. **Charts**: Modify the Chart.js configurations in js/main.js
