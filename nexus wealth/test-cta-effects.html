How the Effects Work

Behind the scenes explanation of the CSS animations.

Primary CTA Effects (.cta-primary)

  • Shiny animation effect that sweeps across the button on hover
  • Subtle 2px lift effect using transform: translateY(-2px)
  • Enhanced shadow effect for depth perception
  • Smooth 0.3s transition for all animations

Secondary CTA Effects (.cta-secondary)

  • Subtle shiny animation effect for border-style buttons
  • 2px lift effect on hover
  • Shadow enhancement for better visual feedback
  • Color transition effects for background and text

Form Button Effects (.form-btn)

  • Simple but effective 2px lift effect
  • Shadow enhancement for better click feedback
  • Optimized for form submission buttons
  • Consistent with overall design language
