Mobile Menu Test

Resize your browser to mobile width or use mobile device to test the hamburger menu.

The mobile menu should:

  • Toggle open/close when clicking the hamburger icon
  • Change hamburger icon to X when open
  • Close when clicking outside the menu
  • Close when clicking on menu links
  • Close when resizing to desktop width
  • Have smooth animations
