# ConstructPro - Professional Construction Website Template

A complete, responsive construction website template built with HTML, Tailwind CSS, and vanilla JavaScript. This template includes all requested pages with professional design, animations, and RTL support.

## Features

- **Responsive Design**: Works on all devices from mobile to desktop
- **Professional Color Scheme**: Uses a premium 4-color palette (primary: #1a365d, secondary: #2c5282, accent: #e53e3e, light: #f7fafc)
- **RTL Support**: Global RTL toggle button for dynamic layout switching
- **Smooth Animations**: Hover effects, transitions, and entrance animations
- **Complete Page Set**:
  - Home Version 1 (index.html)
  - Home Version 2 (index2.html)
  - Projects Page (projects.html)
  - About Page (about.html)
  - Contact Page (contact.html)
  - Admin Dashboard (admin-dashboard.html)
  - User Dashboard (user-dashboard.html)
  - 404 Page (404.html)
  - Coming Soon Page (coming-soon.html)

## Pages Overview

### Home Pages (index.html & index2.html)
- Hero sections with call-to-action buttons
- Featured projects showcase
- Services overview with icons
- Client testimonials with ratings
- News/blog previews
- Contact information sections

### Projects Page (projects.html)
- Grid layout of construction projects
- Filtering system by project type and status
- Project cards with images, descriptions, and details
- Load more functionality

### About Page (about.html)
- Company overview and history timeline
- Mission and values section
- Team member profiles
- Achievements and awards showcase

### Contact Page (contact.html)
- Contact information with icons
- Contact form with validation
- Google Maps integration
- Business hours display

### Admin Dashboard (admin-dashboard.html)
- Dashboard overview with statistics
- Project management system
- Team member management
- Site content editing
- Inquiry tracking
- Blog/news management

### User Dashboard (user-dashboard.html)
- Personal dashboard with project overview
- User project tracking
- Inquiry management
- Profile settings

### Special Pages
- **404.html**: Custom error page with navigation
- **coming-soon.html**: Countdown timer with email signup

## Technical Features

- **Tailwind CSS**: For responsive styling and utility classes
- **Font Awesome**: For icons throughout the site
- **Vanilla JavaScript**: For interactivity and animations
- **Mobile-First Design**: Fully responsive layout
- **Accessible**: Proper semantic HTML and ARIA attributes
- **Performance Optimized**: Efficient code structure

## Installation

1. Download or clone the repository
2. Open any HTML file directly in a browser
3. No build process required

## Customization

- **Colors**: Modify the color palette in the `<script>` section of each HTML file
- **Content**: Edit the HTML content directly
- **Images**: Replace placeholder images with your own
- **RTL**: Toggle RTL layout using the globe icon in the header
