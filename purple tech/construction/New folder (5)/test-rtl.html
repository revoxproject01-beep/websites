How RTL Toggle Works

The RTL toggle button (globe icon) in the top navigation bar allows you to switch the entire website layout between Left-to-Right (LTR) and Right-to-Left (RTL) directions.

This feature is especially useful for languages that are written from right to left, such as Arabic, Hebrew, and Urdu.

LTR Layout (Default)

Text flows from left to right. This is the standard layout for English and many other languages.

RTL Layout

Text flows from right to left. This layout is used for languages like Arabic and Hebrew.
