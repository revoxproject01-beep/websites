How Navigation Highlighting Works

  • The JavaScript in js/nav-highlight.js automatically detects the current page
  • It adds the nav-active class to the matching navigation link
  • Dropdown menu items get the dropdown-active class when active
  • Special handling for related pages (e.g., index2.html also highlights the Home link)
  • Dropdown buttons are highlighted for dashboard and home pages

Expected Behavior

  • On about.html: The "About" link should be highlighted

  • On faq.html: The "FAQ" link should be highlighted
