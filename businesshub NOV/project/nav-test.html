Navigation Highlighting Test

This page tests the navigation highlighting functionality. The current page link in the navigation should be highlighted.

How Navigation Highlighting Works

  • The JavaScript code in js/nav-highlight.js automatically detects the current page
  • It adds the nav-active class to the matching navigation link
  • Dropdown menu items get the dropdown-active class when active
  • Special handling for related pages (e.g., index2.html also highlights the Home link)
